# Reasoning Framework

This framework defines how the system evaluates text for:
- emotional tone
- deception indicators
- reasoning drift
- logical stability

## 1. Emotional Tone
Evaluates:
- sentiment direction
- intensity
- emotional shifts across the text

## 2. Deception Indicators
Evaluates:
- misleading phrasing
- hidden assumptions
- persuasive framing
- risk level

## 3. Reasoning Drift
Evaluates:
- topic drift
- emotional drift
- logic drift
- stability score

## 4. Integration
Each module returns structured data.
A higher-level evaluator can combine these signals into a unified assessment.
